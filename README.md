##Simple Notes SPA App with AngularJS
==============================================================

This is a simple fully functional SPA application.
This application using AngularJS as an end-to-end solution on
front-end. 
Using Angular $routeProvider to set rotes defenition.

Using $resource service to create resource object that
interacts with RESTful server-side data sources that was
implemented using ASP.NET WebApi. 

Data saved into local DB by Entity Framework.
